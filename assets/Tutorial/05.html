Panic Mode Whatsapp

Panic Mode also can send message through Whatsapp.

Change the Panic mode message medium in Settings to Whatsapp.

After starting Panic Mode, WhatsApp is opened. Your supposed to select a broadcast/group/contact to send the Emergency message.

Unlike SMS way, Messages cannot be sent to multiple contacts at once.

It is better to sent the message to a Broadcast/group.

Create a Broadcast ,with all the members you want to send emergency message, in whatsapp and name it FeelSafe . when you start Panic Mode select FeelSafe Broadcast and emergency messages are sent to them.

There is no interval for whatsapp mode, the Messages are sent only once. Start panic mode again to send messages again.
